│ # MMR Tracker for OBS
│ We need something to reference when the state of Rocket League changes. Once we havethat, we just use the built in `txt-trigger.lua` script to turn on a source's visibility based on our reference.
│
│ ## RocketStats Plugin for Reference Value
│ Install the RocketStats plugin for bakkesmod. Available as plugin `30` or here: https://bakkesplugins.com/plugins/view/30 . This plugin will generate and maintain text files that we will reference later.
│
│ ### Finding Reference Value
│ The text file we will use is `RocketStats_BoostState.txt` and is located in the path `C:\Users\kyle\AppData\Roaming\bakkesmod\bakkesmod\data\RocketStats`. In order to find your `AppData` folder, open a windows file explorer window, click in the address bar, and type in `%appdata%`. After hitting enter, you should be in your `AppData\Roaming` folder. The text file has the contents `-1` when in the main menu, and `0` when controlling the car (in game or training). This means that we can match to either `-1` or `0` depending on if we wanta source visible in the menu, or in games.
│
│ ## Creating the Tracker
│ Either go to https://tracker.gg/overlays to create your own overlay, or feel free to use the ones I created:
│
│ - pdg-2v2-tracker source :: https://tracker.gg/overlays/overlay/68609077-e484-4ec7-bae0-6e2692cb737d
│ - pdg-2v2-tracker editor :: https://tracker.gg/overlays/editor/68609077-e484-4ec7-bae0-6e2692cb737d
│ - fivearms-2v2-tracker source :: https://tracker.gg/overlays/overlay/c02ea716-9cb6-4804-80e4-a3aea48857da
│ - fivearms-2v2-tracker editor :: https://tracker.gg/overlays/editor/c02ea716-9cb6-4804-80e4-a3aea48857da
│
│ Note that by making your browser source for this tracker in OBS have width x height of 290 x 90 should crop down to just the tracking information.
│
│ ## Putting it Together in OBS
│ Here weneed to:
│ 1. Insert the sources we care about as **top-level sources** (no nesting them in folders or groups!).
│ 2. Create a copy of the `txt-trigger.lua` script for us to use.
│ 3. Customize our script in OBS
│
│ ### Add Sources in OBS
│ In the scene that you want to use, add the sources (trackers, labels, etc.) as top-level sources. (**No source groups!**)
│
│ ### Create a Copy of the Script
│ Make sure to close OBS completely, then navigate to `C:\Program Files\obs-studio\data\obs-plugins\frontend-tools\scripts` in a windows file explorer window. There, you should see a file `txt-trigger.lua`. Simply copy and paste the file to make a duplicate of it. You may need to agree to elevating your permissions to edit files in this directory. Name the copy whatever you like, I suggest `txt-trigger-tracker.lua`.
│
│ ### Customize the Script in OBS
│ LaunchOBS. In the top menu bar, choose `Tools` then `Scripts` from the drop-down. Below the `Loaded Scripts` window, you should see a `+` button. Clickthe `+` button to add the duplicate script you made previously. Now that it is listed in the `Loaded Scripts` window, click on it and we will change its settings. To the right, you should see settings like `File to check`, `Trigger check period`, etc.
│
│ - File to check :: click `Browse`, then type `%appdata` in the address bar, then navigate to `bakkesmod\bakkesmod\data\RocketStats\`, then select the file `RocketStats_BoostState.txt`
│ - Trigger check period :: 1000 (This is how often the file ischecked by the script, and I think checking every second is plenty often.)
│ - Trigger on any change in the file contents :: **NO** check
│ - Trigger when file contents match pattern :: `^-1` (This is a regular expression pattern that says, "match when the first characters are `-1`". Check how regex patterns work at https://regex101.com .)
│ - Make source(s) visible for as long as file contents match :: **YES** check
│ - Source visibility duration :: not applicable
│ - Source visibility delay :: 0
│ - Source 1 (2, or 3) :: select your tracker, label or other source
│ - Affect all scenes :: **NO** check
│ - Only affect scene :: select your RocketLeague scene
│
│ # Enjoy!
│ At this point, your tracker should be visible when in Rocket League menus, and not visible when in game. Enjoy having an automatically toggling tracker!
│
│ If you wanted to have, say,your controller overlay work the same way, but be visible at opposite times, simply repeat this process, but change the matching pattern for thescript from `-1` to `^[0-9]` (matching any time the first character of the file is a digit 0-9).
